Tellus Coffea
=============

Procedural world generation in CoffeeScript, inspired by Sean O'Neil's [A Real-Time Procedural Universe](http://www.gamasutra.com/view/feature/131507/a_realtime_procedural_universe_.php?page=1).

* main.html --- Planets.
* 2D.html --- 2D terrain.
* 1D.html --- 1D terrain.
